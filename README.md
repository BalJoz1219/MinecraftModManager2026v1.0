Help: 

First: You should run it when the Windows Defender popped up. You can check the .py if you are cautious, its not a virus, even tho the Widows Defender says it protected your computer.

Second: You have to give the manager the mod's path folder via the Change mod folder path button, the mods folder is
in the .minecraft folder, you can access it with Windows+R, than type %appdata%, into the .minecraft folder,
than Control+Shift+C, it should look like this: "C:\\Your_Files\\AppData\\.minecraft\\mods", you should delete the strings on the front and on the end.

If you want to upload a mod into the manager you should use the path to the mod, you can do  it with Shift+Control+C, it should look like this when you copy
it into the input: "C:\\Your_Files\\Downloads\\The_Mod", you should delete the two strings to add it.

You can make a pack, by clicking the button and nameing it as you want. Than add the mods.

You can view all your packs, by clicking the All Packs button.

You can add mods to a pack by clicking on the below the All mods, and clicking the Add to pack button.

You can remove them from a pack the same way.

If you cant add a mod, click the new pack button and write the name of the existing pack to the New Packs name slot, than add the mod via path.

You can delete a pack by clicking the button and giving the name.

You can delete a mod by giving the path via the Delete mod button.

To put your pack into Minecraft you should click the Choose Pack button than give the name of the pack. (You can put more than one pack in.)

To Unchoose a pack, click the button and give the name of it.
